What is Micro-Templating?

What is a micro-templator?

Why would you need to use one?

Defintion

template
something that serves as a model for others to copy
a preset format for a document or file, used so that the format does not have to be recreated each time it is used

Template Processing: A Brief History

template processor: software designed to combine one or more templates with a data model to produce one or more resulting documents. This can be server-side, client-side, edge-side, outside the server, or even in distributed systems.

Templating is opinionated! Meaning everyone and their dog has created a templating engine and they all think their's is the best.

Typical Features of Templators

  1. variables and functions
  2. text replacement
  3. file inclusion (transclusion)
  4. conditional evaluation and loops

  8.     // load the template - and be sure to cache the result.

  10.       cache[str] = cache[str] ||
  11.         tmpl(document.getElementById(str).innerHTML) :
  12.      
  13.       // Generate a reusable function that will serve as a template
  14.       // generator (and which will be cached).
  15.       new Function("obj",
  16.         "var p=[],print=function(){p.push.apply(p,arguments);};" +
  17.        
  18.         // Introduce the data as local variables using with(){}
  19.         "with(obj){p.push('" +
  20.        
  21.         // Convert the template into pure JavaScript
  22.         str
  23.           .replace(/[\r\t\n]/g, " ")
  24.           .split("<%").join("\t")
  25.           .replace(/((^|%>)[^\t]*)'/g, "$1\r")
  26.           .replace(/\t=(.*?)%>/g, "',$1,'")
  27.           .split("\t").join("');")
  28.           .split("%>").join("p.push('")
  29.           .split("\r").join("\\'")
  30.       + "');}return p.join('');");
  31.    
  32.     // Provide some basic currying to the user
  33.     return data ? fn( data ) : fn;
  34.   };
  35. })();
